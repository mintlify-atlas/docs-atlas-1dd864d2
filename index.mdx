## Get started in 3 steps

<Steps>
  <Step title="Browse available layers">
    Visit [mappipeline.com](https://mappipeline.com) or the [layers overview](/layers/overview) to find the data you need.
  </Step>
  <Step title="Copy the KML URL">
    Each layer has a direct raw URL. Copy it from the web interface or the documentation.

    ```
    https://raw.githubusercontent.com/TannerTunstall/MapPipeline/main/kmls/safeairspace-warnings.kml
    ```
  </Step>
  <Step title="Import into your application">
    Paste the URL into your GIS software. The data updates automatically — no need to re-import.

    - **Google Earth Pro**: File → Open → Enter URL
    - **QGIS**: Layer → Add Layer → Add Vector Layer → Protocol: HTTP(S)
    - **ArcGIS**: Add Data → Data From Path
    - **ForeFlight / Garmin Pilot**: Add as custom map layer
  </Step>
</Steps>
