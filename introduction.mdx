## What is MapPipeline?

MapPipeline is a free, open-source service that continuously fetches geospatial data from various sources, converts it into KML format, and makes it available via direct URLs. You can import any layer directly into your preferred GIS software without managing the data pipeline yourself.

KML files are updated on automated schedules — some hourly, some every 5 minutes — so your maps always reflect the latest data.

## How it works

```
Data Sources  →  GitHub Actions  →  KML Files  →  GIS Software
(APIs, sites)    (hourly cron)      (GitHub repo)   (direct URL import)
```

1. **GitHub Actions** runs on a schedule to fetch the latest data from external sources
2. **Node.js scripts** scrape, parse, and transform that data into valid KML format
3. **KML files** are committed to the repository with updated content
4. **Raw GitHub URLs** serve files directly — paste the URL into any GIS application to get live-updating data

## Key features

- **Auto-updating layers** — Aviation warnings, rocket alerts, and event data refreshed automatically
- **Direct URL import** — Each KML file has a permanent URL you can bookmark in your GIS software
- **Interactive preview** — The [web interface](https://mappipeline.com) lets you preview any layer on an interactive map before importing
- **No account required** — All KML files are publicly accessible with no API key or login
- **Free hosting** — Powered by GitHub and Cloudflare Pages at zero cost
