description: "Import your first live KML layer in under 2 minutes — no account or API key required."
---

MapPipeline KML files are served via direct GitHub raw URLs. You import the URL once, and your GIS application automatically gets the latest data every time it loads.

## Available layers

| Layer | Direct URL | Update Frequency |
|-------|-----------|-----------------|
| SafeAirspace Warnings | `https://raw.githubusercontent.com/TannerTunstall/MapPipeline/main/kmls/safeairspace-warnings.kml` | Hourly |
| Israel Rocket Alerts | `https://raw.githubusercontent.com/TannerTunstall/MapPipeline/main/kmls/israel-alerts.kml` | Every 5 minutes |
| ICE Protest Events | `https://raw.githubusercontent.com/TannerTunstall/MapPipeline/main/kmls/mobilize-events.kml` | Hourly |

<Steps>
  <Step title="Find the layer you need">
    Browse the [Available Layers](/layers/overview) page to find the data layer that suits your use case.

    - **Aviation & drone ops** → [SafeAirspace Warnings](/layers/safeairspace-warnings)
    - **Situational awareness in Israel** → [Israel Rocket Alerts](/layers/israel-alerts)
    - **Civic event tracking** → [ICE Protest Events](/layers/mobilize-events)
  </Step>

  <Step title="Copy the KML URL">
    All KML files follow this URL pattern:

    ```
    https://raw.githubusercontent.com/TannerTunstall/MapPipeline/main/kmls/<filename>.kml
    ```

    For example, to get the SafeAirspace aviation warnings:

    ```
    https://raw.githubusercontent.com/TannerTunstall/MapPipeline/main/kmls/safeairspace-warnings.kml
    ```

    <Note>
      This URL is permanent. The file content updates automatically on schedule — you only need to import it once.
    </Note>

    You can also copy any URL from the [web interface](https://mappipeline.com) using the **Copy URL** button on each layer card.
  </Step>

  <Step title="Import into your GIS application">
    Paste the URL into your preferred application:

    <Tabs>
      <Tab title="Google Earth Pro">
        1. Open **Google Earth Pro**
        2. Go to **File → Open**
        3. In the file dialog, enter the KML URL directly into the filename field
        4. Click **Open**

        The layer will appear in your **Places** panel and auto-refresh when you reopen the file.
      </Tab>
      <Tab title="QGIS">
        1. Open **QGIS**
        2. Go to **Layer → Add Layer → Add Vector Layer**
        3. Set **Source Type** to `Protocol: HTTP(S), cloud, etc.`
        4. Paste the KML URL into the **URI** field
        5. Click **Add**
      </Tab>
      <Tab title="ArcGIS">
        **ArcGIS Pro:**
        1. On the **Map** tab, click **Add Data**
        2. Select **Data From Path**
        3. Paste the KML URL and click **Add**

        **ArcGIS Online:**
        1. In the **Map Viewer**, click **Add → Add Layer from Web**
        2. Select **A KML file**
        3. Paste the KML URL and click **Add Layer**
      </Tab>
      <Tab title="ForeFlight / Garmin Pilot">
        **ForeFlight:**
        1. Go to **Maps → Content → Custom**
        2. Tap **+** to add a new layer
        3. Select **KML/KMZ file** and paste the URL

        **Garmin Pilot:**
        1. Go to **Map Options → Custom Layers**
        2. Tap **Add Custom Layer**
        3. Enter the KML URL and save
      </Tab>
    </Tabs>
  </Step>
</Steps>
